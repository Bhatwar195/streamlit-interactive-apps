# Streamlit Interactive Apps

A collection of **practical Streamlit applications** demonstrating interactive widgets, charts, file handling, data filtering, machine learning examples, and basic text processing using Python.

These apps are designed for learning Streamlit and showcasing interactive data applications.

---

## Features

- Interactive UI components (text input, sliders, buttons)
- Data visualization (native charts, Matplotlib, Altair)
- File upload and download
- Caching and progress indicators
- Dashboard layout and multi-column views
- Machine learning demos
- Text processing and wordcloud visualization

---
## Technologies Used

- Python
- Streamlit
- Pandas, NumPy
- Matplotlib, Altair
- Scikit-learn
